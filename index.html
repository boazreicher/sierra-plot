Sierra Plot#

Sierra Plot is a Grafana panel plugin for visualizing datasets containing a very large number of series
A Sierra Plot is an extension of the Ridgeline Plot1, with a number of additional elements, most notably:

  • An added chart showing the totals across all of the different series
  • Fog for highlighting values above some threshold
  • Grouping
  • And much more

  1. Although the use case is quite different, the original motivation is Brendan Gregg's Frequency Trails ↩
